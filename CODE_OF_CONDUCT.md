# Code of Conduct

## Our commitment

This project welcomes contributions and discussion from anyone, and expects participants to
treat each other with respect regardless of experience level, background, or identity.

## Expected behavior

- Be respectful and constructive in issues, pull requests, and discussions.
- Assume good faith; disagree about code and ideas, not about people.
- Give and accept constructive feedback gracefully.
- Focus on what's best for the project and its users.

## Unacceptable behavior

- Harassment, insults, or personal attacks of any kind.
- Publishing others' private information without consent.
- Sustained disruption of discussions or reviews.
- Any conduct that would reasonably be considered inappropriate in a professional setting.

## Scope

This applies to all project spaces — issues, pull requests, code review comments, and any other
official communication channels for this repository.

## Enforcement

Instances of unacceptable behavior may be reported by opening a GitHub issue or, for sensitive
matters, contacting the maintainer directly through their GitHub profile
(https://github.com/andrewsferreira). Reports will be reviewed and addressed as appropriate,
which may include a warning or removal from project spaces.

## Attribution

This Code of Conduct is a short, project-specific adaptation of the spirit of the widely-used
[Contributor Covenant](https://www.contributor-covenant.org/), not a verbatim reproduction of
it.
